## Overview

A fullstack developer portfolio showcasing projects, skills, and professional experience. Built with **React** and **TailwindCSS** for a clean, modern presentation.

## Features

- Project showcase with filters
- Skills and tech stack display
- Professional experience timeline
- Contact form
- Responsive design
- Dark/light theme

## Tech Stack

| Technology | Purpose |
|---|---|
| **React** | Frontend framework |
| **TailwindCSS** | Styling |
| **Node.js** | Backend |
| **Docker** | Containerization |

## Getting Started

```bash
git clone https://github.com/rafaumeu/portfolio_dev.git
cd portfolio_dev
npm install
npm run dev
```
